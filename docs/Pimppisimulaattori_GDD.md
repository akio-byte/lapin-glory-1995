# Pimppisimulaattori / Lapin Glory – Game Design Document (Canon)

## 1. Core Concept
Lyhyt selitys: Lama-Noir / 1995 Rovaniemi, markan ajan selviytymispeli, 
päivä–yö–aamu -looppi, kolme resurssia, EU-faxit, turistit, shamaanit, ankea huumori.

## 2. Resources (Canon)
- **Rahat (mk)**
- **Maine**
- **Järki (Mielenterveys)**

## 3. Stats + Game Over Conditions
- Rahat < -1000 mk → Voudin huutokauppa
- Järki <= 0 → Suljettu osasto
- Maine > 95 → Veropetos-ratsia
- Päivä 30 → Vappu-endingit (tilanneperusteinen)

## 4. Day–Night–Morning Loop
- Päivä: byrokratia, faxit, shop
- Yö: kohtaamiset, baarit, turistit
- Aamu: raportti ja seuraavan päivän modifikaattorit

## 5. Skill Checks
- PIMPPAUS
- BYROSLAVIA
- SISU

## 6. Core Event Themes
- EU-faxit
- Kurkkudirektiivi
- Verotarkastaja
- Turistit ja shamaanit
- Lapin myyttiset ilmiöt
- GSM / Nokia 2110 Net Monitor

## 7. Boss Encounters
- Verotarkastaja / Paper War
- Staalo / Metsänpeitto-episodit

## 8. Uniqueness
- Suomen markka
- Psykologinen eristyneisyys
- Bureaucracy-horror + surrealismi
